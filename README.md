# Mapa de Congregaciones en Uruguay

Este proyecto es un **mapa interactivo** de Uruguay que muestra todas las congregaciones, indicando:

- Nombre de la congregación
- Total de publicadores extranjeros
- Número de publicadores con necesidad de apoyo

El mapa está creado con **Leaflet.js** y los datos se cargan desde un archivo JSON (`data.json`). Los marcadores cambian de color según la cantidad de personas con necesidad de apoyo:

- Verde: 0
- Naranja: 1-2
- Rojo: 3 o más
